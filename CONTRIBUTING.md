# Contributing

Thanks for your interest in improving RiskAPI examples!

## What to contribute

Issues and feature requests are welcome, including:
- Example improvements (Node / Python / C#)
- New language examples (Go, Rust, Java, etc.)
- Better onboarding (README clarity, curl examples, Postman tips)
- Bug reports (wrong URL, missing header, outdated output examples)

## How to report a bug

Please open an issue and include:
- What you tried (request / code)
- What you expected
- What you got (response + status code)
- Any relevant logs or error messages
